WebDMX

WebDMX

WebDMX is a beginner-friendly programming editor for designing interactive stage lighting.

Conducting user tests with artists and musicians informed me that while WebDMX made it easy to use physical sensors and controls to trigger lighting changes, it was still difficult to programming lighting animations. Even when making simple lighting animations, users ended up with extremely long programs that were hard to debug. Participants expressed wanting a more "visual" way of planning out their animations.
